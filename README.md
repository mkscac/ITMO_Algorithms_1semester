Laboratory work numbers:

2. Sorting algorithms

    - merge sort 
3. Stack and queue
4. Binary search tree
5. AVL
6. Coding algorithms
    - arithmetic coding
    - Hamming
    - code Huffman code
    - Gray's codes
    - LZ 78
7. Dynamic programming
